Nash Topbar Test Page

This page tests the nash-topbar block. The topbar should be fixed to the top of the viewport at 52px height, showing:

  • Adobe logo mark (red square + triangle) + "Nash" wordmark + "/ AEM Qualifier" subtitle
  • Current page title centred ("Overview" by default)
  • "New Analysis" primary red button (right side)
  • "VG" user avatar circle (red tint, right side)

Clicking "New Analysis" dispatches a nash:new-analysis CustomEvent on document.

Other blocks can update the centred page title via: document.dispatchEvent(new CustomEvent('nash:page-title', { detail: { title: 'New title' } }))
